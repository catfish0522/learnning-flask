## About Flask

Flask是一个微内核的Web开发框架。

Flask主要依赖两个库：

- [Jinja2](https://palletsprojects.com/p/jinja/)，模板引擎。
- [Werkzeug](https://palletsprojects.com/p/werkzeug/)，一个WSGI套件。

如何理解微内核呢？文档里是这样说的：
> The "micro" in microframework means Flask aims to keep the core simple but extensible.

内核精简，易于扩展，功能不失强大。
